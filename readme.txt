A wordpress plugin that lets you send Free SMS to your subscribers using your Way2SMS account without leaving wordpress admin area.

== Description ==

*Toll Free SMS* is a wordpress plugin that lets you send Free SMS to your subscribers using your Way2SMS account without leaving wordpress admin area.

You must have a valid Way2SMS account for using this plugin.

Check out the screenshots.

= Localization =

* English - (en_EN)

== Installation ==

1. Upload `Toll Free SMS` plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the `Toll Free SMS`Menu under Dashboard Menu and login with your Way2SMS account and start sending sms.

== Frequently Asked Questions ==

= 1. Does it cost anything to me? =
No. As We are using the Way2SMS account for this, it won't cost a single penny to you.

= 2. Can I register Way2SMS account using this plugin? =
No. You must register and activate the Way2SMS accoun, for using this plugin. Its almost free.
